# 单票系统字段与模块总契约表 V1

## 0 总则
- 版本：V1
- 作用：锁字段、完整模块边界
- 主键：`symbol + trade_date`
- 时间：`YYYY-MM-DD`
- 口径稳定：字段定义冻结后不得漂移
- 输入输出边界：数据仓只输出标准化字段与质量状态

## 1 数据层（Data Layer）
- raw：原始行情、指数、事件、公司行为
- clean：标准化字段 + 固定口径
- base：基础衍生字段（收益率/波动/价差）
- contract：字段契约文档版本- 主表必备字段：symbol、tra- 主表必备字段：symbol、trade_date、open/high/low/close、volume、amount

- 分类：价格趋势、波动、量价、市场联动、事件窗口
- 元信息：字段名、定义、缺失处理、标准化、版本

## 2 标签层（Label Layer）
- 目标：统一y定义，避免不同模块各算各的
- 分类标签：上涨、下跌、大涨、大跌、波动放大、regime切换
- 回归标签：y_close、y_ret；horizon：T+1/T+5/T+20/T+120
- 分位标签：用于P05~P95学习与区间覆盖校准
